# 🤖 Jarvis v4.0 - ONE-CLICK INSTALL

**Download → Double-Click → Done!**

## 🚀 Quick Start (No Terminal Needed!)

### Option 1: One-Click Installer (RECOMMENDED)
```
1. Download this repo as ZIP
2. Extract
3. Double-click: Jarvis_OneClick_Setup.bat
4. Paste your API key
5. Done!
```

### Option 2: GUI Installer
```
1. Download this repo as ZIP
2. Extract
3. Double-click: jarvis_gui_installer.py
4. Paste API key in window
5. Click Install
```

## 🆘 Troubleshooting

**"Python not found"**: Run `Jarvis_OneClick_Setup.bat` - it will install Python automatically!

**"Can't access from phone"**: Make sure you're on the same WiFi. Check your firewall.

**"API key error"**: Make sure you pasted the FULL key starting with "AI..."
